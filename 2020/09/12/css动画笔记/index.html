记录学习动画中的知识点

flex布局：

使目标居中: 

display: flex; 
justify-content: center;
align-items: center;

animation布局:

用@keyframes定义一个函数名字: 

@keyframes hd{
    from{
            background: white;

     }
    to{
           background: #e74c3c;

    }
}

然后用于目标: 

 animation-name: hd;
 animation-duration: 2s;

最后差不多这样子，别的效果可以自己另外加入: 

 @keyframes hd{
0%{
    background: white;

}
50%{
    transform: scale(2);

}
100%{
    background: #e74c3c;

}
}

移动的小方快:
先设置一个大盒子main，在里设置一个div，

 @keyframes hd{
25%{
    transform: translateX(300px);

}
50%{
    transform: translate(300px,300px);
    <!-- 移动方向距离 -->

}
100%{
    transform: translateY(300px);


}
}

main和div的 

main{
width: 400px;
height: 400px;
background: white;

}  
div{
width: 100px;
height: 100px;
background: yellow;
animation-name: hd;
border: solid 1px #ddd;
animation-duration: 2s;
}

影响

属性重叠对动画影响:

加入定义三个动画，分别是translate,animation,border.

分别有不同的影响,但是时间分别是4s,2s,1s.

优先级从后面往前面，所以1s先border，2s的animation剩下1s，所以translate剩下2s。

transform可以旋转，缩放，移动，倾斜

scale()可以缩放大小。

::after是伪类元素，就是加入一些特别的小动画。

animation-iteration-count: 控制次数.

背景可以有渐变选项。

动画也有方向选择。

section是阴影，可以用。
filter是阴影的虚实。
比如说一个小球跳起来的影子。


动画延迟的作用:因为动画的时间包含延迟时间运用填充css.
贝塞尔控制曲线速率: https://cubic-bezier.com/
